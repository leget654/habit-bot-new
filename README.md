# 🔥 Habit Tracker Bot

Telegram-бот для отслеживания привычек с inline-кнопками, стриками, геймификацией, мини-приложением и социальной системой.

## Возможности

### ✅ Базовые
- Отмечать привычки кнопками каждый день
- 🔥 Стрики — счётчик дней подряд
- 📊 Статистика: прогресс-бар, процент выполнения за месяц
- 📅 Календарь-сетка как на GitHub (✅ = выполнено)
- ⏰ Напоминания в выбранное время
- ➕ Добавлять / удалять / переименовывать привычки с эмодзи

### 🔥 Геймификация и мотивация
- ❄️ **Streak Freeze** — заморозка стрика за вчера (1/неделю на каждую привычку)
- 🌟 **Бонус «Идеальный день»** — +30 XP когда все привычки за день выполнены
- 🎯 **Ежедневные квесты** — «Отметь 3 привычки до 12:00» → +20 XP
- 🌅 **Мотивационные цитаты** — при открытии бота и каждый день в 8:00
- 🏅 **Расширенные достижения** — за стрики, количество привычек, квесты, жаворонок/сова, habit stacking

### 📊 Статистика и аналитика
- 💡 **Инсайты** — «Ты наиболее продуктивен по вторникам», «Самый длинный стрик — зарядка (45 дн.)», «Средняя завершённость 78%»
- 📅 **Навигация по месяцам** ◀️ Месяц ▶️
- 📈 **% выполнения по каждой привычке** за неделю и месяц
- 📤 **Экспорт данных** в JSON, CSV, PNG-карточка для шеринга
- 🖼 **Карточка месяца** — генерируется через Pillow

### 🏗 Функциональность привычек
- 🏷 **Категории/теги** — Здоровье, Учёба, Работа, Разум, Спорт, Питание, Творчество и др.
- 📋 **Шаблоны привычек** — 15 готовых (Утренняя зарядка, Чтение 30 мин, Медитация и т.д.)
- 🔗 **Habit Stacking** — «после ☕ Кофе → 🏃 Зарядка»
- ⏱ **Таймер/трекер времени** — запуск/стоп с авт-отметкой при достижении цели
- ↕️ **Порядок привычек** — кнопки ▲▼
- ✅ **«Отметить все»** — одной кнопкой
- ↩️ **Отмена действия** — undo последней отметки
- 📆 **Отметка за прошлые дни** — до 30 дней назад

### 👥 Социальные фичи
- 🎁 **Реферальная система** — пригласи друга → +3 дня Premium
- 👥 **Друзья** — заявки, список, последняя активность
- ⚔️ **Челленджи** — 7-дневный вызов между друзьями
- 🔥 **Мотивационные пинги** — «Твой друг в огне! Поздравь его»

### 🛠 Архитектура
- 📦 **Модульная структура** — `db.py`, `keyboards.py`, `constants.py`, `utils.py`, `handlers/`, `services/`
- 💾 **FSMStorage → SQLite** — состояние не теряется при рестарте
- ⏰ **APScheduler** — вместо `asyncio.sleep(60)` цикла
- 🗄 **Индексы в БД** — `completions(habit_id, completed_date)`, `habits(user_id)`, и др.
- 🌍 **Часовые пояса** — для каждого пользователя
- 🚫 **`.gitignore`** — `habits.db` удалён из git

## Быстрый старт

### 1. Получить токен бота

Открой [@BotFather](https://t.me/BotFather) в Telegram:
```
/newbot
```
Скопируй токен вида `7123456789:AAF...`

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Настроить Mini App (опционально)

Хостинг для `webapp/`:
- Railway / Render / Vercel — задай `WEBAPP_URL` env переменную

### 4. Запустить

```bash
BOT_TOKEN=7123456789:AAF... python bot.py
```

## Команды бота

| Команда | Действие |
|---------|----------|
| `/start [ref_CODE]` | Начать, показать меню (с реферальным кодом) |
| `/menu` | Главное меню |
| `/reset` | Сбросить FSM-состояние |
| `/premium` | Экран подписки |
| `/quote` | Случайная мотивационная цитата |

## Деплой

### Railway.app (бесплатно)
1. Залить код на GitHub
2. Подключить репозиторий на [railway.app](https://railway.app)
3. Добавить переменные окружения `BOT_TOKEN`, `WEBAPP_URL`
4. Deploy ✅

### VPS / systemd
```ini
[Unit]
Description=Habit Tracker Bot

[Service]
ExecStart=/usr/bin/python3 /path/to/bot.py
Environment=BOT_TOKEN=ВАШ_ТОКЕН
Environment=WEBAPP_URL=https://your-app.example.com
Restart=always

[Install]
WantedBy=multi-user.target
```

## Архитектурные улучшения

| До | После |
|----|-------|
| `bot.py` 1715 строк | Разбит на модули, `bot.py` ~120 строк |
| `MemoryStorage` (теряет состояние) | `SqliteStorage` (переживает рестарт) |
| `asyncio.sleep(60)` цикл | `APScheduler` с cron-триггерами |
| Без индексов | 6 индексов для производительности |
| Без часовых поясов | Поле `timezone` у каждого пользователя |
| `habits.db` в git | Удалён, `.gitignore` добавлен |
